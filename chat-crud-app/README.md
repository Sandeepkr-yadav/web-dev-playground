# 💬 Chat CRUD App

A simple Chat CRUD application built using **Node.js, Express.js, MongoDB, Mongoose, and EJS**. This project demonstrates basic CRUD (Create, Read, Update, Delete) operations with server-side rendering and MongoDB integration.

## 🚀 Features

* Create new chats
* View all chats
* Edit existing messages
* Delete chats
* Server-side rendering using EJS
* MongoDB database integration with Mongoose
* RESTful routes with Method Override
* Clean and simple UI

## 🛠️ Tech Stack

* Node.js
* Express.js
* MongoDB
* Mongoose
* EJS
* Method Override
* CSS

## ⚙️ Installation

1. Clone the repository

```bash
git clone <repository-url>
```

2. Navigate to the project directory

```bash
cd chat-crud-app
```

3. Install dependencies

```bash
npm install
```

4. Start MongoDB locally.

5. Run the application

```bash
node app.js
```

6. Open your browser and visit:

```
http://localhost:8080/chats
```
